# Linux Mining README

### Currencies
- ZCash (ZEC)
- ZClassic (ZCL)
- Ethereum (ETH)

### Miners
- EWBF
- Claymore10

### First Time Setup
1.) Run 
`./initial_setup` 
`./nvidia_gpu_setup.sh` 
 
2.) Reboot

### Every boot
1.) Run 
`sudo ./overclock.sh` 

2.) Run mining script `./start_xxx.sh` where xxx is currency
